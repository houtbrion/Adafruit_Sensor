# AusExI2cTemplate

このドライバは，
[Adafruitのセンサ統合ライブラリ][AdafruitUSD]を拡張した上で，いろいろなセンサ類を
[統合ライブラリ][AdafruitUSD]のAPIでアクセス可能にするプロジェクトの一部で，
Bosh Sensortecの[BMP180気圧センサ][BMP180]用ドライバです．

動作確認には，[Grove - Barometer Sensor][ProductPage]を用いました．
そのため，[Seeed studio][SeedStudio]，[Adafruit][Adafruit]共に独自の
ドライバを公開していますが，今回の開発では[Adafruitのドライバ][AdafruitGithub]を
改造しました．


## ライセンスについて
改造元のAdafruitのドライバには，明確なライセンス記述がありませんが，
以下のような記述がREADMEの末尾に付記されております．

```
Adafruit invests time and resources providing this open source code. Please support Adafruit and open-source hardware by purchasing products from Adafruit!

Written by Kevin (KTOWN) Townsend for Adafruit Industries.
```

そのため，私のコードについても保留しておきます．
