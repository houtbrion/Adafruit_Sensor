# AusExGroveSimpleLight


このライブラリ(センサ類のドライバ)は，
[Adafruitのセンサ統合ライブラリ][AdafruitUSD]を拡張した上で，いろいろなセンサ類を
[統合ライブラリ][AdafruitUSD]のAPIでアクセス可能にするプロジェクトの一部で，
[Groveアナログ光センサ][ProductPage]用ドライバです．


## 仕様
|機種|バージョン|フォトレジスタ|
|:---|:---|:---|
|Grove - Light Sensor(P)|V1.0|photoresistor-5528|
|Grove - Light Sensor(P)|V1.1|LS06-S|
|Grove - Light Sensor|V1.1まで|photoresistor-5528|
|Grove - Light Sensor|V1.2|LS06-S|

## 使い方
このセンサは特別な測定動作のモード切り替えやセンサの出力の変換も不要なため，特別なAPIは
存在しておらず，getMode()やsetMode()は-1を返すだけの動作となってます．

また，センサの出力値は0～1023の非負整数(uint32_t)であり，これはADコンバータの
出力そのままの値です．
